# 💆‍♀️ Бот-анкета косметолога

Telegram-бот, який збирає анкети клієнтів. Посилання на бота кладете в Instagram (опис профілю / stories) → людина переходить → бот по черзі ставить питання з варіантами відповідей → формує анкету → клієнт підтверджує → анкета прилітає вам у Telegram з номером телефону та @username клієнта для зворотного зв'язку.

---

## 🚀 Швидкий старт (3 кроки)

### Крок 1. Токен бота (у вас уже є)
Токен — це рядок виду `123456789:AAE...` від [@BotFather](https://t.me/BotFather).
Якщо треба новий: у BotFather → `/newbot` → назва → username (має закінчуватися на `bot`) → отримаєте токен.

### Крок 2. Дізнатися свій `ADMIN_CHAT_ID`
Це число, **куди надсилати анкети** (ваш особистий Telegram).
1. Тимчасово впишіть токен (див. нижче) і запустіть бота.
2. Напишіть своєму боту команду **`/whoami`** — він відповість вашим `chat_id`.
3. Це число впишіть у `ADMIN_CHAT_ID`.

> Альтернатива: напишіть боту [@userinfobot](https://t.me/userinfobot) — він теж покаже ваш id.

### Крок 3. Вписати токен і id
Скопіюйте `.env.example` у `.env` і заповніть:
```
BOT_TOKEN=сюди_ваш_токен
ADMIN_CHAT_ID=сюди_ваш_chat_id
```

## ☁️ Деплой у хмару 24/7 (Railway — безкоштовно)

Щоб бот працював завжди, навіть коли комп'ютер вимкнено.

1. Зареєструйтеся на [railway.app](https://railway.app) (через GitHub).
2. Залийте цю папку в репозиторій GitHub **або** натисніть `New Project → Deploy from local`.
3. У проєкті відкрийте вкладку **Variables** і додайте:
   - `BOT_TOKEN` = ваш токен
   - `ADMIN_CHAT_ID` = ваш chat_id
4. Railway сам побачить `requirements.txt` і `Procfile` та запустить `worker: python bot.py`.
5. Готово — бот онлайн 24/7.

> **Render.com** — так само: New → Background Worker → Build `pip install -r requirements.txt`, Start `python bot.py`, у Environment додати ті ж дві змінні.

## 📲 Посилання для Instagram

Посилання на бота: **`https://t.me/ІМ'Я_ВАШОГО_БОТА`**
(те саме, що username бота без `@`).

Куди вставляти:
- в опис профілю Instagram (поле «сайт»);
- у stories через стікер «Посилання»;
- у Linktaplink / Taplink, якщо користуєтесь.

Можна додати мітку джерела: `https://t.me/ІМ'Я_БОТА?start=instagram` — людина все одно одразу потрапляє в анкету.

## ✏️ Як змінити питання

Усі питання — у файлі **`questions.py`**. Там зрозумілі коментарі.
Додати/прибрати питання, поміняти варіанти відповідей чи тексти бота можна без знань програмування — просто редагуйте список `QUESTIONS`. Після зміни перезапустіть бота (локально — Ctrl+C і знову `python bot.py`; у Railway — кнопка Redeploy).

Типи питань:
- `text` — клієнт пише довільну відповідь;
- `choice` — кнопки, один варіант (+ `allow_custom: True`, щоб дозволити свій);
- `multichoice` — кнопки, кілька варіантів + «Готово»;
- `contact` — кнопка «Поділитися номером».

## ⌨️ Команди бота
- `/start` — почати анкету;
- `/cancel` — скасувати заповнення;
- `/whoami` — показати свій chat_id (для налаштування).
